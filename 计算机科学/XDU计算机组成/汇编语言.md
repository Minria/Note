# 指令

## 传送指令

1、传送指令

```assembly
MOV OPRD1, OPRD2
```

2、交换指令

```assembly
XCHG OPRD1,OPRD2
```

3、地址传送指令

```assembly
LEA OPRD1,OPRD2
```

```assembly
LDS SI, [BX]
```

```assembly
LDS SI, [BX]
```

4、堆栈操作指令

```assembly
PUSH OPRD

POP OPRD
```

5、字节、字转换指令

```assembly
MOV AL , 4FD

CBW  ;将字节转换一个字 AX = 004FH
```

```assembly
MOV AX ,834EH
CWD # 字节转换为双字 DXAX = FFFF834H
```

6、标志寄存器指令

```assembly
LAHF
SAHF
PUSHH
POPF
```
